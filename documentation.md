Accessibility
Accessibility is important aspect for website development and I made sure I will include as much as I could in my
website.
First accessibility point which I made was adding a hover into my navigation bar to make sure user will see where to click
in order to get access to rest of the website. So when user move mouse over a navigation bar one of the buttons will
change colour to more lightly.
Another thing which I added is a description of photos and gifs. If the photo will not be able to be displayed it will
display instead a short message describing photo. This feature is included in all of my photos and in slide show. So
people with disability who will not be able to see it at least will be able to hear or read what is about.
As i mention above in colour section I added a hover over download section so user will know that it is a link which
will make some sort of action.
User can navigate through website without mouse. All the website is accessible from keyboard and can be used by space ,
arrows and tab buttons.
Last accessibility i did provide for user is a check when user would like to send a email. I gave a validation which will
change colour to light green when validation is succesful or light red when it fails.
